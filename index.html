jQuery is a JavaScript Library, the purpose of which is to make it easier to write JavaScript for website


jQuery takes a lot of common tasks that requires a many lines of coding to accomplish, and wraps them into
methods that we can call with a single line of code.

jQuery simplies complicated things of JavaScript like AJAX and DOM manipulation.S

jQuery library contains the following features

  • HTML/DOM manipulation
  • CSS manipulation
  • HTML events methods
  • Effects and animation
  • AJAX
  • Utilities

On top of all these, jQuery has many more plugins for different tasks

jQuery basic syntax:

$(selector).action()

  • $ sign to access jQuery
  • A selector to "query (or find)" HTML elements
  • A jQuery action() to be performd on the element

jQuery selectors allow us to select and manipulate html elements.
jQuery selectors are used to find or slect HTML elements based on their name,id,classes,types,values of
attribute and much more
